# shoe-player: A project for Python course @ PCC
## Deliverable (super alpha)
#### Music application with prescribed features:
##### 1. Search for music.
##### 2. Store your music.
##### 3. Like, dislike a tune.
##### 4. Skip a song.
##### 5. Pause.
##### 6. User can create, modify, delete playlist.
